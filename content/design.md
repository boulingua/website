Boulingua is one platform with many languages. To keep every language project
recognisable on its own while still feeling part of the whole, each language has
**its own signature accent colour** and **its own abstract icon**.

## How the colours are chosen

Each accent is:

- **Flag-safe** — the hue is deliberately chosen *not* to appear in that
  language's own national flag, so the mark reads as a boulingua identity rather
  than a flag;
- **Distinct** — every accent is at least 20° apart from every other on the hue
  wheel;
- **Reserved around the hub** — no language uses the boulingua master blue
  (`#1A73E8`), which belongs to this hub and the shared resources;
- **Accessible** — the light value meets WCAG AA contrast on white, and a
  matching lighter value is used on the dark theme.

Each language also has a solid-accent **geometric mark** (hexagon, circle,
square, …) used in its repository README, on this site, and on every slide deck
and worksheet it produces.

The CEFR level ramp (A `#4CAF50`, B `#1A73E8`, C `#8E24AA`) is shared across all
languages and is independent of the per-language accents.

The colour system is the single source of truth in
[`data/accents.yaml`](https://github.com/boulingua/website/blob/main/data/accents.yaml);
the same values drive the websites' CSS and the `slidegen` / `sheetgen` LaTeX
templates.
