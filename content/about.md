Boulingua is a free, openly licensed multilingual teaching platform created by
S. Le Boulanger. It is built for language teachers and learners working in the
German school system — specifically Gesamtschule Baden-Württemberg — and for
international learners of German following the Common European Framework of
Reference (CEFR).

The platform brings together teaching units for three languages — English,
French, and German as a foreign language — alongside a curated hub of free,
openly licensed teaching resources. EFL and FLE units are organised by school
year (Klasse) and aligned to the Bildungsplan Baden-Württemberg, including
Niveau-Differenzierung (G / M / E). The DaF units follow Goethe-Institut
examination formats and are organised by CEFR level (A1 to C1).

All four sister sites share an identical design system, so a teacher or
student moving between languages encounters the same navigation, the same
typography, and the same visual rhythm. Consistency is a feature, not an
accident.

The four content sites are built with [Quarto](https://quarto.org/) and this
landing page is built with [Hugo](https://gohugo.io/). All five repositories
are hosted under the [boulingua GitHub organization](https://github.com/boulingua)
and deployed via GitHub Pages.
